## Professional Experience

### Recruit CRM

At Recruit CRM, I have worked mainly on PHP Laravel-based microservices and backend systems. My work involved building scalable services and improving communication between different parts of the system.

Some of the things I have worked on:

* Developed and maintained microservices using PHP and Laravel
* Implemented queue systems in Laravel for handling asynchronous tasks
* Used Redis for caching and improving performance
* Worked with AWS services like Amazon S3 and AWS SQS
* Contributed to Kafka implementation in Laravel for event-driven communication

One of the key things I built was a custom Laravel package that is used across multiple microservices. This package stores AWS SQS payloads in Amazon S3, which helps overcome the 256 KB payload size limitation of SQS.
